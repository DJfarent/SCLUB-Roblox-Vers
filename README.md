# secretservice.club (fan made)
A clean and stable Roblox script for Criminality using Rayfield UI.

### Features
- **Player Chams** (Real-time color + transparency)
- **Persistent Custom FOV** (Stays even when game tries to change it)
- **Hold-to-Aim Aimbot** (Stiff lock, bindable key)
- **FOV Circle** (Visible when aiming)
- **Hit Notifications** with Rust headshot sound on every hit you make

### Tabs
- **Visuals** → Player Chams (color, fill & outline transparency)
- **Player** → Custom FOV (persistent)
- **Aimbot** → Bind, Aim Part, FOV, Smoothness
- **Misc** → Hit Notifications toggle
